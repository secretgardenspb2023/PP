# Telegram: привязка бота к домену (`/setdomain`)

Чтобы на сайте работала кнопка **«Войти через Telegram»**, бот должен быть привязан к домену,
на котором стоит виджет. Делается один раз в **@BotFather**.

- Бот: **PoiskPlant Login** (`@poiskplant_login_bot`)
- Домен сейчас: **`staging.poiskplant.ru`** *(при запуске боевого сайта поменяем на `poiskplant.ru`)*

> Виджет Telegram работает только на **живом HTTPS-домене** (на `localhost` — нет).
> `staging.poiskplant.ru` уже с HTTPS, так что можно привязывать.

---

## Способ A — через меню (рекомендую, без ошибок)

1. Открой **@BotFather** в Telegram.
2. Отправь команду **`/mybots`**.
3. В списке нажми на бота **@poiskplant_login_bot**.
4. Нажми **«Bot Settings»**.
5. Нажми **«Domain»** (или «Configure Login URL» / «Linked Domain»).
6. Отправь сообщением домен: **`staging.poiskplant.ru`**
   *(без `http://`, без `https://`, без слэша в конце — просто домен).*
7. BotFather ответит **«Success! ...»** — готово.

---

## Способ B — командой `/setdomain`

1. Отправь **`/setdomain`**.
2. BotFather покажет **кнопки с твоими ботами** внизу экрана (клавиатура).
3. **❗ Нажми кнопку `@poiskplant_login_bot`** — НЕ печатай название руками.
4. Когда попросит домен — отправь: **`staging.poiskplant.ru`**

---

## Почему была ошибка «Invalid bot selected»
Ты вводил(а) текст вручную («PoiskPlant Login» или «poiskplant_login_bot»), а BotFather
после `/setdomain` ждёт **нажатие кнопки** бота из появившейся клавиатуры. Решение —
нажать кнопку (способ B, п. 3) или использовать меню (способ A).

## Другие возможные ошибки
- **«Bad domain»** — убери `https://` и слэши, оставь чистый `staging.poiskplant.ru`.
- Не видно кнопок ботов — значит сработал автотекст; используй способ A через `/mybots`.

## Как проверить, что привязалось
- BotFather после ввода домена пишет **«Success!»**.
- Перепроверить: `/mybots` → бот → Bot Settings → Domain — там будет показан текущий домен.
